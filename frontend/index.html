Help Guide

Getting Started

This interface allows you to interact with an explainable AI system. Type your query in the chat box and the system will provide both an answer and a visual explanation of its reasoning process.

Visualization Types

  • Reasoning Tree: Shows the AI's decision-making process as a tree structure.
  • Token Importance: Displays the relative importance of each token in the reasoning.
  • Counterfactual Analysis: Shows how changing specific tokens would affect the outcome.
  • Knowledge Graph: Connects reasoning with external knowledge sources.
  • Divergence Points: Highlights where different reasoning paths diverge.

Interactive Features

  • Click on nodes in the reasoning tree to see detailed information.
  • Adjust node importance to see how it affects the reasoning process.
  • Toggle different reasoning paths to compare alternative explanations.
  • Export visualizations for documentation or further analysis.

Settings

The settings panel allows you to configure various aspects of the AI system:

  • Choose different language models
  • Adjust token generation parameters
  • Configure CGRT (Counterfactual Graph Reasoning Tree) settings
  • Set up counterfactual analysis parameters

About CGRT

CGRT (Counterfactual Graph Reasoning Tree) is a technique for analyzing and visualizing AI reasoning processes. It generates multiple reasoning paths, identifies divergence points, and enables counterfactual analysis to understand how different inputs would affect the output.
